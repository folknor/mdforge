# Installation

## CLI

Install globally with npm:

```bash
npm install -g mdforge
```

Or with pnpm:

```bash
pnpm add -g mdforge
```

## Desktop App

Download the portable desktop app for your platform from the [releases page](https://github.com/folknor/mdforge/releases).

| Platform | Format |
|----------|--------|
| Windows | `.exe` (portable, no installer) |
| Linux | AppImage |
| macOS | `.app` bundle |

## Requirements

- Node.js 24 or later (for CLI)
- Chromium is bundled with Puppeteer (downloaded automatically)
