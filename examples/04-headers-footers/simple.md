# Simple Footer

This example shows a basic centered footer with page numbers.

## How It Works

The `footer` option accepts a simple string:

```yaml
footer: Page {page} of {pages}
```

## Available Variables

- `{page}` - Current page number
- `{pages}` - Total page count
- `{title}` - Document title (from first h1)
- `{date}` - Current date
